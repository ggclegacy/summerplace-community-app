# Summer Place Community Vision

An interactive presentation/demo site for the Summer Place resident experience app concept, created by Gent Ascend Collective.

## Stack
- React 18 + Vite 6
- Tailwind CSS 3
- Framer Motion 11
- Lucide React (icons)

## Running the app
```
npm run dev
```
Runs on **port 5000**. The Replit workflow `Start application` handles this automatically.

## Project structure
```
src/
  App.jsx     — full single-page app (all sections)
  main.jsx    — React entry point
  styles.css  — global styles
index.html    — HTML shell
vite.config.js — Vite config (host 0.0.0.0, port 5000)
```
